常用入门git指令

当前目录内创建新仓库
>git init

克隆远端仓库（GitHub）
>git clone $远端仓库地址

添加改动到缓存区
>git add *

提交改动至本地仓库
>git commit -m “提交代码”

推送改动至远端仓库
>git push origin master

创建并切换到分支
>git checkout -b feature_X

切换回主分支
>git checkout master

删除新建分支
>git branch -d feature_X

更新本地仓库
>git pull
